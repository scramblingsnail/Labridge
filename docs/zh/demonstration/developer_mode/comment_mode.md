# 在 Acting phase 评论
如果你打开评论模式，你可以在Labridge的 Acting phase 对ta的动作所得到的observation进行评论，Labridge会参考你的评论进行下一步决策。

## 评论模式设置
在这个例子中，我们同时打开了[指令模式](instruct_mode.md)与评论模式。
